This folder stores historical internal documents preserved for traceability.

## Archiving criteria

Move a doc from `../stable/` to `archive/` when:

- feature design has been implemented and released,
- analysis investigation has reached a final conclusion,
- report has been superseded by newer standards.

Each archived markdown file should include front matter keys:

- `status: archived`
- `archived_date`
- `archived_from`
- `archive_reason`
