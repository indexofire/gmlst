# Stable Internal Docs

Use this area for active internal documents that guide current implementation.

## Placement rules

- Ongoing design proposals
- Active investigation notes with unresolved decisions
- Internal standards that are still being evolved

## Naming conventions

- Design: `<topic>_design.md`
- Analysis: `<topic>_analysis.md`
- Reports/standards: `<topic>_report.md` or `ADR-<n>-<topic>.md`

When a document is complete and no longer actively maintained, move it to
`../archive/`.

## Starter templates

- `ongoing_architecture_report.md` — active architecture tracking template
- `archive_migration_report.md` — checklist for moving docs to archive
